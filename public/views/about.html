Acerca de

Nuestra intención es facilitar el uso de Bibtex para generar documentos en latex y otros.

Usando este servicio buscamos dar el mejor servicio posible. Los registros se obtienen desde interfaces publicas provistas por The International DOI Foundation y Crossref. Sin embargo el BibTeX podría contener errores y sugerimos siempre revisarlos.

Reconocimientos

El código base para este sitio es open source y lo puedes encontrar en GitHub.
